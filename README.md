# ⚓ Lastro

> **Equilíbrio pras suas finanças.**

Lastro é um app de controle de gastos pessoais com foco em um visual agradável, atraente e de fácil entendimento. Você registra seus gastos e o Lastro organiza tudo, mostrando pra onde seu dinheiro vai de um jeito que faz sentido.

## O nome

- ⚓ **Náutico:** o lastro é o peso que dá estabilidade ao navio pra ele não virar.
- 💰 **Financeiro:** "ter lastro" é ter reserva e garantia — segurança e base sólida.
- 🏗️ A ideia central: um bom alicerce antes de mais nada.

## Plataformas

- 📱 **App mobile Android** (foco inicial)
- 🌐 **Site / aplicação web**
